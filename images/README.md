# images/

Esta carpeta almacena las imágenes que se usarán en las diapositivas de la presentación.

## Convención de nombres sugerida

| Archivo              | Descripción                              |
|----------------------|------------------------------------------|
| `slide1-bg.png`      | Imagen de fondo / portada Slide 1        |
| `slide2-network.png` | Mapa de red global (Slide 2 – Internet)  |
| `slide3-*.png`       | Imágenes adicionales Slide 3             |
| `logo-ufps.png`      | Logotipo oficial UFPS                    |
| `logo-sis.png`       | Logotipo Ingeniería de Sistemas          |

> Coloca aquí tus archivos `.png`, `.jpg`, `.svg` o `.webp` y
> referencialos en las diapositivas con la ruta relativa `images/nombre.ext`.
